Exercise tracker

Description

This service allows to create users and assign exercises to them.
The information about stored users and exercises assigned to them can be easily retrieved using our API!

Create a new user

Get a list of all users

https://impartial-taker.glitch.me/api/exercise/users

Add exercises

* - required field

Get user's exercise log

API call

/api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional
from, to = dates (yyyy-mm-dd); limit = number

Example Usage

https://impartial-taker.glitch.me/api/exercise/log?userId=xkuZUKjCo&from=2019-03-01&to=2019-04-15&limit=2
